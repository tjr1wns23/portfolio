칸반보드

📝 프로젝트 개요

Vue 3, TypeScript, Vite를 활용해 제작한 칸반 보드 애플리케이션입니다. 작업을 To Do, In Progress, Done의 세 영역으로 구분해 진행 상황을 한눈에 확인할 수 있으며, 마우스 드래그 앤 드롭 기능을 통해 편리하게 작업을 관리할 수 있습니다.

🛠️ 사용 기술

  • Vue 3
  • TypeScript
  • Vite
  • nodeJS

📌 주요 기능

  • 새 일정 등록(추가)기능
  • 각 일정 드래그앤 드랍 기능
  • 진행과정 내 순서 재배치 기능

🔍 개발 포인트

  • TypeScript 타입 지정

    TypeScript의 정적 타입 지정과 타입 추론을 활용하며, 안정성과 가독성을 높이는 코드 작성 방법을 학습할 수 있었습니다.

  • 컴포넌트 설계

    Vue 컴포넌트를 활용한 모듈화로, 재사용성과 유지보수성이 뛰어난 코드 구조를 설계·구현하는 방법을 학습하였습니다.

  • vite 빌드/배포

    Node.js와 Vite를 활용해 프로젝트 구조를 설계하고 배포용 빌드까지 구현하였습니다.

💭 느낀 점

컴포넌트를 활용해 유지보수가 용이한 코드 구조를 설계하며, 컴포넌트 관리의 장점을 체감할 수 있었습니다. 또한 DataTransfer를 활용해 컴포넌트 간 데이터 이동을 효과적으로 처리하며 관련 작업에 익숙해질 수 있었습니다. transition-group을 사용해 부드러운 애니메이션 효과를 간단히 구현하면서 Vue가 제공하는 기본 기능의 편리함을 느꼈고, 앞으로도 이러한 내장 기능을 적극적으로 활용해야겠다는 생각이 들었습니다.

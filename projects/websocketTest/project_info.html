🗨️ 웹소켓 채팅

📝 프로젝트 개요

Websocket을 이용해 구현한 채팅 어플리케이션입니다. 닉네임 설정, 알람끄기 기능 등을 구현했으며, 실시간 채팅이 가능하도록 구현했습니다.

🛠️ 사용 기술

  • Websocket
  • NodeJS

📌 주요 기능

  • 라이브 채팅
  • 새 메시지 도착 시document.title에 알람
  • 닉네임 설정

🔍 개발 포인트

  • Websocket

    Websocket 기능을 활용해 라이브 채팅이 가능한 실시간 통신 기능을 구현하였습니다.

  • 유저메시지, 서버메시지 구분

    WebSocket은 기본적으로 텍스트 기반 통신만 지원하기 때문에, 알림 메시지와 유저 채팅 메시지를 구분하기 위해 JSON 포맷을 사용해 데이터를 전송하고, 클라이언트에서 파싱하여 처리했습니다.

  • 에러 처리

    서버 측 에러 발생 시 개발자가 쉽게 원인을 파악할 수 있도록, WebSocket의 에러 메시지를 콘솔에 명확하게 출력하는 기능을 추가했습니다.

💭 느낀 점

기본적인 형태의 채팅 애플리케이션을 구현하며, 2000년대의 채팅 서비스(예: 돛단배, 네이트온)를 연상시키는 레트로한 UI를 만드는 과정이 꽤 재미있었습니다. 또한 WebSocket의 단순한 텍스트 통신 구조 속에서 JSON을 활용한 메시지 타입 구분 로직을 설계하며, 데이터 인코딩 및 파싱에 대한 이해도를 높일 수 있는 좋은 경험이었습니다.

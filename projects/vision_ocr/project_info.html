길드 관리 콘솔

📝 프로젝트 개요

Google Vision API 기반의 길드 관리 콘솔로, 길드 점수 이미지를 업로드하면 OCR을 통해 자동으로 텍스트 데이터를 추출하고, 이를 변수화하여 지난 2주간 점수를 효율적으로 합산하고 평균을 계산할 수 있도록 개발했습니다.

🛠️ 사용 기술

  • Google Vision API
  • nodeJS

📌 주요 기능

  • 클라이언트 사이드 이미지 업로드
  • 이미지에서 선택 영역 지정
  • 이미지 -> 텍스트 추출
  • 텍스트 데이터 연산 처리

🔍 개발 포인트

  • Google Vision API

    Google Vision OCR키의 노출을 방지하기 위해, 서버 사이드에서만 키를 보관하고 이미지 업로드 시 서버에서 직접 텍스트 변환을 수행하도록 구현하여 안전하고 효율적인 OCR 기능을 제공했습니다.

  • 지정 영역 변환 처리

    OCR 처리 시 불필요한 데이터가 포함되는 문제와 자동 변환 과정에서 발생할 수 있는 오타를 최소화하기 위해, 필요한 영역만 선택적으로 변환하도록 구현하여 가독성과 정확성을 높였습니다.

  • 배열 데이터 관리

    추출한 데이터를 배열로 저장하여 지난 컨텐츠에 모두 참여한 인원, 1회만 참여한 인원, 미참여 인원등의 정보와 2주간의 평균점수를 한눈에 확인할 수 있도록 데이터 처리 기능을 개발했습니다.

💭 느낀 점

2주간의 길드 컨텐츠 점수 정보를 합산하여 길드 승강전을 진행하는 과정에서, 매번 인게임 데이터를 직접 계산하는 번거로움을 해결하기 위해 이미지에서 점수를 자동으로 추출하는 기능을 개발했습니다. 이 과정에서 Google Vision OCR API를 서버사이드에서 연동하여 처리하도록 구현함으로써 키 보안을 유지하고, 서버-클라이언트 역할을 명확히 분리하는 방법을 익혔습니다.

이 페이지는 서버가 구동되어야 작동하므로 직접 해보기가 불가합니다.

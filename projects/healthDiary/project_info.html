💪 운동일지

📝 프로젝트 개요

Vue 3 CDN을 활용해 개발한 SFA 개인 운동 기록용 어플입니다. 사용자가 날짜별로 운동 내용을 작성하고, 기록을 관리할 수 있는 기능을 구현했습니다.

🛠️ 사용 기술

  • Vue 3 CDN
  • JavaScript (Composition API)

📌 주요 기능

  • 홈 화면 동기부여 멘트 랜덤 표출
  • 날짜별 운동 기록 등록/수정/삭제
  • 홈/일지 보기/일지 작성 페이지 구성

🔍 개발 포인트

  • Composition API

    vue watch 기능을 활용해 반응형 상태값을 관리하며, 코드 가독성을 신경썼습니다.

  • 상태 공유 및 데이터 흐름 체계적 구성

    다뤄야 하는 많은 변수들을 객체(object)로 구조화하여 설계하고 데이터 흐름을 체계적으로 관리하는 데 집중했습니다.

💭 느낀 점

데이터를 설계하며 변수의 재사용성과 유지보수를 고려한 개발을 연습할 수 있었습니다.
Vue에서는 HTML 내에 변수를 바인딩하여 사용하기 때문에, 마크업만으로도 기능을 쉽게 파악할 수 있어 유지보수가 쉬워지는 코드를 작성하는 데 집중했습니다.
이 과정을 통해 데이터 관리의 중요성과 구조적인 코드 작성의 필요성을 체감할 수 있었습니다.

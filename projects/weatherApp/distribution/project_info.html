날씨 앱

📝 프로젝트 개요

React, TypeScript, Vite를 활용해 제작한 날씨 앱입니다. OpenWeatherMap API를 이용해 지역별 날씨 정보를 가져오며, 하루 날씨와 일주일간 예보를 한눈에 확인할 수 있도록 구성했습니다. 또한 직관적인 아이콘 표시를 통해 사용자가 쉽게 날씨 정보를 이해할 수 있도록 구현했습니다.

🛠️ 사용 기술

  • React
  • TypeScript
  • Chart.js
  • vite
  • OpenWeatherMap API

📌 주요 기능

  • 오늘 날씨, 미세먼지, 대기질 지수 등 정보 처리
  • 지역 변경 기능
  • 주간 날씨 예보 기능
  • 오늘 날씨 그래프로 직관적 시각화

🔍 개발 포인트

  • TSX 컴포넌트 설계

    React와 TypeScript를 활용하여 타입 안전성과 컴포넌트 구조화를 학습하고, 재사용 가능한 TSX 컴포넌트를 설계했습니다.

  • 유틸리티 함수/커스텀 훅

    TypeScript의 유틸리티 함수와 커스텀 훅을 체계적으로 구성하고 활용하며, React + TypeScript에서 컴포넌트 생명주기와 상태 관리 방법에 대한 이해를 심화할 수 있었습니다.

  • OpenWeatherMap API

    OpenWeatherMap API를 연동하며 API 통신 처리와 데이터 가공에 익숙해질 수 있었습니다.

💭 느낀 점

이번 프로젝트에서 React와 TSX를 처음 사용하며, React의 생명주기와 훅을 학습하면서 한층 성장할 수 있었습니다. 또한 날씨 정보를 다루는 TypeScript 인터페이스 구조를 설계하며, 데이터 가공 능력과 타입 설계 역량을 향상시킬 수 있었습니다. 이번 경험을 바탕으로 앞으로도 지속적으로 학습하며 더 깊이 있는 개발 역량을 쌓아가고자 합니다.
